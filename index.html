jsConsole

jsConsole is a highly responsive, pure client-side platform for on-the-fly JSON manuiplation. Originally inspired by Underscore.js, it is now powered by the following (ridiculously extensive) technology stack:

  • LoDash - To provide the base functional programming grammar.
  • Underscore-contrib - To extend to some genuinely far-reaching edge cases.
  • Underscore.string - To super-power string manipulations.

To work it, load up a JSON file (or just paste it in) to the left panel (top on mobile). Then write your function chain. All of the functions in the above libraries are at your disposal. A trivial example is included to get you started. Be sure you ingest the *input* and save the resulting object to *output*. The contents of output will appear automagically in the right pane. From there you can save the manipulator script and output.

jsConsole is a product of FutureCensus.
